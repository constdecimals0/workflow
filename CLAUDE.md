# Workflow

This repo documents how to run a development workflow using the Matt Pocock skills, with a worked example in `example/`.

## Agent skills

### Issue tracker

Issues live in GitHub Issues on `constdecimals0/workflow` via the `gh` CLI; external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default vocabulary — the five canonical roles used verbatim (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
